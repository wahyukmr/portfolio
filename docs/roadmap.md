### Infra locked

- Netlify (official template)

🚫 Intentionally Unused (MVP)

- CMS
- i18n
- Middleware

- Simple 3D animation

- Articles: 3–5 engineering logs

## PHASE 1 – FOUNDATION

### Deployment & Infra

- Netlify deploys successfully
- SSG build is stable

### Routing

- Route structure final
- `/projects`, `/articles`, etc. are live
- Refresh doesn't 404
- Locale-based routing `/en` `/id` is active

### Boundary Engineering

- Feature-based architecture
- No wild logic in the route

### MDX Infrastructure

- Live folder content
- MDX build-time compile
- Dynamic slug rendering
- No runtime fetch

### Rendering Finalization

- SSG for content
- Edge redirect for `/`
- Theme via loader + action
- No SSR for content
- No middleware creep
